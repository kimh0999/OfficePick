오피스픽 소개

🏢 서비스 개요

오피스픽(OfficePick)은 사용자의 현재 위치를 기반으로 주변의 공유오피스를 쉽게 찾고, 비교하고, 예약까지 가능한 웹 서비스입니다.

✨ 주요 기능

  • 카카오 지도 기반의 오피스 위치 시각화
  • 검색 및 필터 기능
  • 오피스 후기 및 별점 등록
  • 날짜별 예약 기능
  • 관리자 페이지로 후기/예약 확인

👨‍💻 개발자 소개

김준오 - Frontend 개발, UI/UX, 카카오 API 연동

김재환 - 기획, 데이터 설계, 사용자 피드백 정리
